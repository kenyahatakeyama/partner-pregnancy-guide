# Partner Pregnancy Guide（パートナー妊娠ガイド）

パートナーのための妊娠・子育て準備ガイドアプリ。妊娠中から産後まで、パートナーとしてできることを週数に合わせて学べます。

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 機能

- **自動計算される妊娠週数** – 出産予定日から現在の週数・日数を自動計算
- **段階的なガイド** – 妊娠初期から後期まで、今この時期に必要な知識をタブで切り替え可能
- **4つのタブで構成**
  - **今日の支え方** – すぐに実行できる3つのアクションと受診の目安
  - **学ぶ** – 妊娠・出産・産後について6つのテーマから選んで学ぶ
  - **準備する** – 出産までの確認事項・産後の生活設計の準備リスト
  - **ふたりのメモ** – 健診での質問メモと夫婦間の対話メモ（ブラウザに自動保存）
- **ローカルストレージ対応** – チェックリスト・メモは全てブラウザに自動保存される（サーバー不要）
- **ダークモード対応** – OS設定に自動対応
- **レスポンシブデザイン** – モバイル・タブレット・デスクトップで最適表示

## 使い方

1. `index.html` をブラウザで開く
2. 出産予定日は現在のアプリに `2026年10月23日` で設定されています
   - 別の日付に変更する場合は、コード内の `new Date(2026, 9, 23)` を修正してください
3. 各タブで内容を確認、チェックリストの完了をマークする
4. メモを入力して「保存」ボタンを押す

## カスタマイズ

### 出産予定日の変更

`index.html` 内の以下を変更してください：

```javascript
// 39行目あたり
const dueDate = new Date(2026, 9, 23); // 月は 0-11（10月 = 9）
```

また、HTMLの表示部分も合わせて更新してください：

```html
<!-- 100行目あたり -->
<strong>2026年10月23日（金）</strong>
```

### 色のカスタマイズ

CSS変数 (`:root` セクション) で色スキームを変更できます。
