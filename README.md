# ふたりの妊娠ガイド

夫婦ふたりで使う妊娠・子育て準備アプリ。夫モード / 妻モードを切り替えて、それぞれに合ったガイドと、ふたりで共有するメモ・お願いリストが使えます。

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 主な機能

### 👶 赤ちゃんの設定（誰でも使える）
- 出産予定日・性別・名前を設定（ふたりのスマホに同期）
- 名前が決まる前は「名前候補リスト」でふたりで出し合い、お互いに♥を付けられる

### 📅 週数の自動計算
- 出産予定日から妊娠週数・カウントダウンを自動計算
- 1週ごとの見通しトラック（安定期・後期・臨月・正期産のマイルストーン表示）
- 次のマイルストーンまでの日数表示

### 🍉 今週の赤ちゃん
- 週ごとの大きさ（果物のたとえ）・身長・体重の目安
- 発育のひとことトピック、◀▶で他の週も閲覧可能

### ☀️ 毎日変わるコンテンツ
- 「今日できる3つ」：時期（初期/中期/後期）に合わせて毎日ちがうタスクが届く
- 「今日の声かけ例」「今夜の会話のタネ」も日替わり
- 連続チェックイン日数（🔥ストリーク）

### 👨👩 夫モード / 妻モード
- **夫モード**：今日の支え方、準備チェックリスト（進捗バー付き）
- **妻モード**：セルフケア、胎動カウンター、からだと気持ちのメモ、夫へのお願いリスト
- 妻が追加したお願いは夫モードにリアルタイムで届き、完了チェックできる

### 🔄 ふたりのスマホでリアルタイム同期
- 「合言葉」を1つ作って相手のスマホで入力するだけでペアリング
- Firebase Realtime Database 使用（無料枠）。オフライン時はローカル保存で動作

### 💾 自動保存
- すべての入力は打った瞬間に保存（保存ボタン不要）

## 使い方

1. `index.html` をブラウザで開く（または GitHub Pages のURLへアクセス）
2. **👶 赤ちゃん**タブで出産予定日を設定
3. 画面下の「ふたりのスマホで同期」から合言葉を作成し、パートナーのスマホで入力

## 技術仕様

- 単一HTMLファイル（フレームワークなし・ビルド不要）
- データ保存：localStorage ＋ Firebase Realtime Database（同期時）
- 対応：モダンブラウザ全般、レスポンシブ、ダークモード対応

## 注意

- 発育の数値・週数はあくまで一般的な目安です。気になることは必ず産院・医師に相談してください。
- 合言葉を知っている人はデータを閲覧できます。夫婦の間だけで共有してください。
