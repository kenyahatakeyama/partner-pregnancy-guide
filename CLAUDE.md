# Partner Pregnancy Guide

パートナー妊娠ガイドプロジェクト

## プロジェクト概要

妊娠パートナー向けの学習・チェックリスト統合アプリ。出産予定日から現在の妊娠週数を自動計算し、その時期に必要なガイダンスを提供する。

- **形式**: シングルファイルHTML（サーバーレス）
- **技術**: バニラJS + CSS Grid/Flexbox
- **ストレージ**: localStorage（ブラウザ内蔵）
- **対応言語**: 日本語

## ファイル構成

```
index.html        # メインアプリ（HTML + CSS + JS 統合）
README.md         # 使用ガイド・カスタマイズ方法
.gitignore        # Git除外ファイル
CLAUDE.md         # このファイル
```

## 機能

1. **自動週数計算** – 出産予定日 → 現在の妊娠週数・カウントダウン
2. **4タブ構成**
   - 今日の支え方: 日々のアクション + 受診目安
   - 学ぶ: 6つのテーマ（からだ・気持ち・出産・新生児・制度・役割）
   - 準備する: 出産前後チェックリスト
   - メモ: 健診質問・夫婦対話の記録
3. **完全オフライン** – サーバー不要、全データローカルストレージ保存

## 出産予定日の設定

`index.html` の JavaScript部分（~41行目）で以下を変更：

```javascript
const dueDate = new Date(2026, 9, 23); // 2026年10月23日の例
```

括弧内の数値: `new Date(年, 月-1, 日)`  
※ JavaScriptの月は 0-11（10月 = 9）
